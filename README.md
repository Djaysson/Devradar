# Devradar
<p>Projeto da Semana Omnistack #10 da Rocketseat</p>
<p>O projeto Devradar visa conectar desenvolvedores com clientes por meio da localidade, em um raio de 10km.</p>

<h2>Tecnologias usadas</h2>
<ul>
  <li>Node.js</li>
  <li>React</li>
  <li>React Native</li>
</ul>
